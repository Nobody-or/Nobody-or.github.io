Selection sort

Selection sort is a simple and intuitive sorting algorithm. It works by finding the i-th smallest element at a time and swapping that element with the element at position i of the array.

This is the example code of selection sort.

Selection sort


void selection_sort(int* a, int n) {
    for (int i = 1; i < n; ++i) {
        int ith = i;
        for (int j = i + 1; j <= n; ++j) {
            if (a[j] < a[ith]) {
                ith = j;
            }
        }
        std::swap(a[i], a[ith]);
    }
}
