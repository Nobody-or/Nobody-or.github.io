Queue

A queue is a table with the property that "the element that enters the queue first must be queued first". Because of this nature, queues are also commonly referred to as first-in-first-out tables, or FIFO tables for short.

Array simulation queue

Usually an array is used to simulate a queue, and two variables are used to mark the head and tail of the queue. 


int q[SIZE], ql = 1, qr;
//The code corresponding to the queue operation is as follows:
            
q[++qr] = x;//Insert an element
ql++;//Delete element
q[ql];//Access to the head of the queue
q[qr];//Accessing the end of the queue
qr = 0;//Empty the queue: ql = 1

Queues in C++ STL

C++ provides a container std::queue in STL, which requires the <queue> header file to be introduced before use.
The queue container in STL provides a bunch of member functions to call. Some of the more commonly used ones are:
Element access
q.front() returns the first element of the queue
q.back() returns the end of the queue
Modification
q.push() inserts element at end of queue
q.pop() pops the first element of the queue
Capacity
q.empty() whether the queue is empty or not
q.size() returns the number of elements in the queue
