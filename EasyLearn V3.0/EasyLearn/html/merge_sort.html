Merge sort

The merge sort is an efficient comparison-based stable sorting algorithm.It is based on the idea of partitioning an array into segments and then merging them.

Merge sort


        void merge(const int *a, size_t aLen, const int *b, size_t bLen, int *c) {
            size_t i = 0, j = 0, k = 0;
            while (i < aLen && j < bLen) {
                if (b[j] < a[i]) { // Judge b[j] < a[i] first to ensure stability.
                    c[k] = b[j];
                    ++j;
                } else {
                    c[k] = a[i];
                    ++i;
                }
                ++k;
            }
        //At this point, one array is empty and the other is non-empty, merge the non-empty array into c
            for (; i < aLen; ++i, ++k) c[k] = a[i];
            for (; j < bLen; ++j, ++k) c[k] = b[j];
        }
