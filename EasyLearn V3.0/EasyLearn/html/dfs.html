Depth-First Search

Depth First Search (DFS): an algorithm for traversing or searching a tree or graph. Traverse the nodes along the depth of the tree and search the branches of the tree as deeply as possible. When all the edges of node v have been explored or the node does not satisfy the condition at the time of search, the search goes back to the starting node of the edge where node v was found. The process is repeated until all nodes have been visited.

The basic writing method of dfs is recursive, the following is the sample code of the classic dfs topic "All Arrangements".

Example - All Arrangements

Taking any m (m ≤ n) elements from n different elements and arranging them in a certain order is called a permutation of m elements from n different elements. All permutations when m=n are called full permutations.

DFS - All Arrangements


int n;// Array to store the results
int path[N]; // Array used to indicate whether an element has been used or not.
bool st[N]; // array to mark whether an element has been used or not
            
void dfs(int u) {// Output this if dfs layer == max depth
    if (u == n) {
        for (int i = 0; i < n; i++)
            printf("%d ", path[i]);
        return;
    }
     // Depth-first traversal
    for (int i = 1; i <= n; i++) {// Check for duplicates at each element traversed
        if (!st[i]) {
            // Set the value of the current position
            path[u] = i; // mark this value as duplicated.
            st[i] = true; // continue traversing down the path; st[u] = i; // mark this value as used
            dfs(u + 1);// revert to the original
            st[i] = false; }
        }
    }
