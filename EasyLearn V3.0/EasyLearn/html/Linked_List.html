Linked List

A linked list is a data structure used to store data, connecting elements through pointers like a chain. It is characterized by the convenience of inserting and deleting data, but it is not good at finding and reading data.

Constructing a linked list

Unidirectional linked table contains data field and pointer field where data field is used to store data and pointer field is used to connect the current node to the next node. 


struct Node {
    int value;
    Node *next;
};

Inserting (writing) data into a linked table

The process is roughly as follows:
Ⅰ. Initialize the data to be inserted node;

Ⅱ. Point node's next pointer to p's next node;

Ⅲ. Point the next pointer of p to node.


void insertNode(int i, Node *p) {
    Node *node = new Node;
    node->value = i;
    node->next = p->next;
    p->next = node;
}
